# CLI-e-commerce

Homework to create a ecommerce in command line interface using Nodejs

## Run Locally

Before anything, you need to have [node](http://nodejs.org/) and [npm](https://www.npmjs.org/) installed.

    ```sh
    $ npm install
    ```

Then execute bin:

    ```sh
    $ node bin/ecommerce.js
    ```
