# node design patterns

Example of a use some design patterns in nodejs console application

## Run Locally

Before anything, you need to have [node](http://nodejs.org/) and [npm](https://www.npmjs.org/) installed.

```sh
$ npm install
```

Then execute bin:

```sh
$ npm start
```
